# 培训知识库筛选方案

> 目标：从38篇文档中选出18篇以内，采用"先道后术"的培训顺序

---

## 一、筛选原则

### 1.1 先道后术
- **道层**：建立认知框架，理解WHY（6-8篇）
- **术层**：掌握方法技巧，学会HOW（10-12篇）

### 1.2 选择标准
1. **可读性优先**：说人话、有启发、容易读懂
2. **实用性优先**：可操作、有案例、能应用
3. **核心性优先**：核心理论、关键方法、基础框架

### 1.3 排除标准
- ❌ 过于学术化，需要专业背景
- ❌ 格式混乱，需要大量润色
- ❌ 索引性质，不适合直接阅读
- ❌ 内容重复，已有更好版本

## 三、培训路径设计

### 3.1 第一阶段：道层培训（2周，6篇）

**目标**：建立正确的认知框架和价值观

**学习序列**：
1. **心智路径依赖理论** → 诊断：理解认知困境的根源
2. **认知内共生理论** → 方向：确立正确的人机协同路径
3. **构建北极星** → 目标：明确自己的方向
4. **人生的意义和做法** → 价值观：对齐北极星和铅垂线
5. **时间的朋友** → 长期：培养长期主义思维
6. **你的世界在等你按下回车键** → 行动：从等待到主动

**培训方式**：
- 每周3篇，精读+讨论
- 每篇后写反思：如何应用到工作中
- 最后总结：我的认知框架是什么

---

### 3.2 第二阶段：术层培训（3周，12篇）

**目标**：掌握实用的方法和技巧

#### 第1周：沟通技巧（3篇）
1. **不要正面回答别人的问题** → 用户思维
2. **让对方HI起来** → 心理机制
3. **透过现象看本质** → 洞察力

#### 第2周：创新和执行（6篇）
4. **创新三元法** → 创新框架
5. **最小可行启动点** → 启动方法
6. **初创企业生存法则** → 生存逻辑
7. **马斯克五步工作法** → 执行方法
8. **动态决策框架** → 决策方法
9. **思考的模型** → 思维框架

#### 第3周：行为改变和长期策略（4篇）
10. **微习惯和正反馈** → 行为改变
11. **为什么给儿子讲道理时你很清醒，给客户报价时你就短视了** → 知行合一
12. **当真诚成为最狡猾的策略** → 长期策略
13. **当美好变成理所当然** → 保持初心

**培训方式**：
- 每天1篇，精读+实践
- 每篇后设计应用场景
- 最后总结：我的方法工具箱

4. **讨论交流**：团队讨论加深理解
5. **定期回顾**：学完后定期回顾和总结
