# 03_洞察库

本目录存放**跨案例洞察**与**行业趋势**。  
洞察不是口号：必须可追溯、可证伪、可迭代。

---

## 目录结构

- `跨案例洞察/`：从多个案例中抽出的机制规律
- `行业趋势/`：更偏宏观/行业演进的趋势判断（也要写证据链与可证伪点）
- `洞察总索引.md`：全库入口（必维护）

---

## 新增一条洞察的标准流程

1. 用 `00_系统指令/Claude-洞察归纳提示词.md` 输出“洞察卡”
2. 落盘到对应目录（跨案例洞察/行业趋势）
3. 在洞察文档中列出支撑案例（内部链接）
4. 更新 `洞察总索引.md`
5. 回填到支撑案例文档（“相关洞察”）
