# 02_模型库

本目录存放从案例中抽象出的**可复用商业模型**（战略/运营/财务）。  
目标：让模型具备“机制清晰、变量明确、边界清楚、可验证、可迭代”五件套。

---

## 目录结构

- `战略模型/`：竞争、定位、护城河、创新、增长战略等
- `运营模型/`：飞轮、网络效应、增长环、供给-需求匹配等
- `财务模型/`：单位经济、LTV/CAC、现金流、杠杆等
- `模型总索引.md`：全库模型入口（必维护）

---

## 新增一个模型的标准流程

1. 用 `00_系统指令/Claude-模型提炼提示词.md` 输出“模型卡”
2. 选择类型目录并落盘：`战略模型/` 或 `运营模型/` 或 `财务模型/`
3. 在模型文档中列出“支撑案例”（内部链接）
4. 更新 `模型总索引.md`
5. 回填到相关案例文档的“模型标签/相关链接”
