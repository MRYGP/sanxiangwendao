# 文档润色框架模板

> 基于《当真诚成为最狡猾的策略》的成功写法
> 这是后续所有文档润色的标准框架

---

## 零、标题命名法（第一步，最重要）

### 核心原则

**标题是文档的第一印象，决定了读者是否愿意打开**

### 优秀标题的特征

#### 1. 反常识/制造冲突感 ⭐⭐⭐⭐⭐

**原理**：反常识的标题能瞬间抓住注意力，让人想一探究竟

**示例**：
- ✅ **《当真诚成为最狡猾的策略》** - "真诚"和"最狡猾"形成强烈对比
- ✅ **《不要正面回答别人的问题》** - 反常识，让人好奇
- ✅ **《为什么给儿子讲道理时你很清醒，给客户报价时你就短视了》** - 制造冲突感

**命名模板**：
```
[正面词] + 成为/变成 + [负面词/反义词]
[不要/别] + [常规做法]
[为什么] + [看似矛盾的现象]
```

#### 2. 暗示核心观点 ⭐⭐⭐⭐⭐

**原理**：标题就暗示了文章的核心观点，让读者有预期

**示例**：
- ✅ **《当真诚成为最狡猾的策略》** - 暗示"真诚可能是最聪明的策略"
- ✅ **《让对方HI起来》** - 暗示"让对方感到被需要、有价值"
- ✅ **《时间的朋友》** - 暗示"长期主义、复利效应"

**命名模板**：
```
[核心观点] + [比喻/形象化表达]
[动作] + [对象] + [结果]
[概念] + 的 + [比喻]
```

#### 3. 制造悬念 ⭐⭐⭐⭐

**原理**：让人想知道答案，产生阅读欲望

**示例**：
- ✅ **《为什么给儿子讲道理时你很清醒，给客户报价时你就短视了》** - 为什么会有这种矛盾？
- ✅ **《当真诚成为最狡猾的策略》** - 为什么真诚会是最狡猾的？

**命名模板**：
```
为什么 + [矛盾现象]
当 + [A] + 成为 + [B]
[问题] + 的 + [答案/真相]
```

#### 4. 简洁有力 ⭐⭐⭐⭐

**原理**：短标题更容易记忆和传播

**示例**：
- ✅ **《让对方HI起来》** - 5个字，朗朗上口
- ✅ **《时间的朋友》** - 4个字，简洁有力
- ✅ **《构建北极星》** - 4个字，形象生动

**命名模板**：
```
[核心概念] + [动作/状态]
[比喻] + 的 + [对象]
[动作] + [目标]
```

#### 5. 有画面感 ⭐⭐⭐

**原理**：形象化的标题更容易理解和记忆

**示例**：
- ✅ **《让对方HI起来》** - 有画面感，让人联想到兴奋状态
- ✅ **《时间的朋友》** - 拟人化，有画面感
- ✅ **《构建北极星》** - 有画面感，让人联想到方向

**命名模板**：
```
[动作] + [形象化对象]
[比喻] + [动作/状态]
[场景] + [结果]
```

### 标题命名检查清单

#### 必须满足（至少2项）

- [ ] **是否有反常识/冲突感？** - 能瞬间抓住注意力
- [ ] **是否暗示核心观点？** - 让读者有预期
- [ ] **是否制造悬念？** - 让人想读下去

#### 尽量满足（至少1项）

- [ ] **是否简洁有力？** - 10个字以内最佳
- [ ] **是否有画面感？** - 形象化表达
- [ ] **是否朗朗上口？** - 容易记忆和传播

### 标题命名技巧总结

#### 技巧1：反常识法

**公式**：`[正面词] + 成为/变成 + [负面词/反义词]`

**示例**：
- 《当真诚成为最狡猾的策略》
- 《当老实成为最聪明的选择》
- 《当失败成为最好的老师》

#### 技巧2：问题法

**公式**：`为什么 + [矛盾现象/反常识现象]`

**示例**：
- 《为什么给儿子讲道理时你很清醒，给客户报价时你就短视了》
- 《为什么聪明人经常做出愚蠢决策》
- 《为什么越努力越失败》

#### 技巧3：否定法

**公式**：`不要/别 + [常规做法]`

**示例**：
- 《不要正面回答别人的问题》
- 《别做完美主义者》
- 《不要试图说服所有人》

#### 技巧4：比喻法

**公式**：`[核心概念] + 的 + [比喻]`

**示例**：
- 《时间的朋友》
- 《认知的拐杖》
- 《信任的复利》

#### 技巧5：动作法

**公式**：`[动作] + [对象] + [结果]`

**示例**：
- 《让对方HI起来》
- 《构建北极星》
- 《突破潜意识给自己画的那个圈》

### 标题命名常见错误

#### ❌ 错误1：过于学术化

**示例**：
- ❌ 《认知内共生理论：AI时代人类能力进化的根本路径选择》
- ✅ 《当AI成为你的认知教练》

#### ❌ 错误2：过于平淡

**示例**：
- ❌ 《如何建立信任》
- ✅ 《当真诚成为最狡猾的策略》

#### ❌ 错误3：过于冗长

**示例**：
- ❌ 《为什么给儿子讲道理时你很清醒，给客户报价时你就短视了，以及如何解决这个问题》
- ✅ 《为什么给儿子讲道理时你很清醒，给客户报价时你就短视了》

#### ❌ 错误4：没有冲突感

**示例**：
- ❌ 《真诚的重要性》
- ✅ 《当真诚成为最狡猾的策略》

### 标题润色流程

#### 第一步：提取核心观点

**问题**：这篇文章的核心观点是什么？

**示例**：
- 核心观点：真诚可能是最聪明的策略
- 核心观点：不要直接回答问题，要理解对方真实需求

#### 第二步：寻找反常识角度

**问题**：这个观点有什么反常识的地方？

**示例**：
- 反常识：真诚 vs 最狡猾（冲突感）
- 反常识：不要正面回答（反常规做法）

#### 第三步：选择命名技巧

**问题**：用哪种技巧最能表达这个反常识？

**示例**：
- 反常识法：《当真诚成为最狡猾的策略》
- 问题法：《为什么给儿子讲道理时你很清醒，给客户报价时你就短视了》

#### 第四步：检查标题质量

**问题**：
- 是否有反常识/冲突感？
- 是否暗示核心观点？
- 是否制造悬念？
- 是否简洁有力？
- 是否有画面感？

### 标题命名示例对比

#### 示例1：认知内共生理论

**原标题**：
- ❌ 《认知内共生理论：AI时代人类能力进化的根本路径选择》- 过于学术化

**优化方案**：
- ✅ 《当AI成为你的认知教练》- 反常识法，有画面感
- ✅ 《为什么AI用多了会变傻》- 问题法，制造悬念
- ✅ 《AI是拐杖还是教练》- 对比法，制造冲突

#### 示例2：最小可行启动点

**原标题**：
- ⚠️ 《最小可行启动点》- 平淡，没有冲突感

**优化方案**：
- ✅ 《钉子汤的智慧：如何用一颗钉子做一锅汤》- 比喻法，有画面感
- ✅ 《为什么不需要万事俱备才开始》- 问题法，制造悬念
- ✅ 《当一颗钉子成为启动点》- 反常识法，有画面感

#### 示例3：动态决策框架

**原标题**：
- ❌ 《动态决策框架迹象验证驾驭AI创业的不确定性》- 过于冗长

**优化方案**：
- ✅ 《为什么预测不如验证》- 问题法，制造冲突
- ✅ 《当预测成为假设》- 反常识法，简洁有力
- ✅ 《像侦探一样做决策》- 比喻法，有画面感

## 一、整体结构框架（7步法）

### 结构模板

```
开头：震撼故事（吸引注意力）
  ↓
第一部分：历史对比案例（建立深度）
  ↓
第二部分：时代变化分析（解释为什么）
  ↓
第三部分：个人化案例（增加温度）
  ↓
第四部分：权威案例对比（强化观点）
  ↓
第五部分：可操作方法（提供价值）
  ↓
结尾：回到开头故事（首尾呼应）
```

## 二、开头写法（震撼故事法）

### 核心原则

**用反常识的故事开头，瞬间抓住注意力**

### 写法模板

```
时间 + 地点 + 背景（制造冲突感）
  ↓
具体事件（反常识的行为）
  ↓
数据支撑（具体数字）
  ↓
结果反转（出人意料）
```

### 示例分析

**原文开头**：
```
2011年11月25日。
黑色星期五。全美最疯狂的购物节。
所有品牌都在血拼：梅西百货打出"全场3折起"...
户外品牌Patagonia也在《纽约时报》买了整版广告。
所有人都在等着看他们的促销文案。
广告打开的瞬间，营销总监的手机炸了：
**"Don't Buy This Jacket" （别买这件夹克）**
```

**结构拆解**：
1. **时间+地点+背景**：2011年11月25日，黑色星期五，购物节
2. **制造冲突感**：所有品牌都在促销，Patagonia也买了广告
3. **反常识行为**："别买这件夹克"
4. **数据支撑**：12万美元广告费
5. **结果反转**：销售额暴涨30%

### 应用要点

- ✅ 用具体时间、地点、数据
- ✅ 制造冲突感（反常识）
- ✅ 用数据支撑（12万美元、30%增长）
- ✅ 结果要出人意料

## 三、案例组织法（多维度案例体系）

### 核心原则

**建立多维度、多层次的案例体系**

### 案例维度

#### 1. 历史案例（建立深度）

**写法**：
- 时间 + 地点 + 人物 + 具体细节
- 用数据支撑（100日元、25日元、0.18日元）
- 对比两种截然相反的方式

**示例**：
- 松下幸之助（1907年）：亏到卖被子也要履约
- 菲尔·奈特（1962年）：靠空壳公司一路骗

#### 2. 现代商业案例（强化观点）

**写法**：
- 具体事件 + 时间 + 数据
- 对比成功 vs 失败
- 用数据说话（1200亿美元、740亿美元）

**示例**：
- Facebook（2018年）：数据泄露，1200亿美元蒸发
- 安然（2001年）：破产，740亿美元蒸发
- 巴菲特（2008年）："我也不知道"，300%回报

#### 3. 个人化案例（增加温度）

**写法**：
- 用身边人的故事
- 具体细节（七岁时、28岁、婚后第三年）
- 用对话和内心独白

**示例**：
- 堂妹的故事：从"人精"到选择"老实"

### 案例组织原则

1. **时间跨度**：从历史到现代
2. **空间跨度**：从日本到美国
3. **人物跨度**：从企业家到普通人
4. **结果跨度**：从成功到失败

## 四、对比论证法（强化观点）

### 核心原则

**用对比让读者思考更深层的问题**

### 对比类型

#### 1. 人物对比

**写法**：
- 两个截然相反的人
- 用相同的问题引出
- 展示不同的选择

**示例**：
```
他们用两种截然相反的方式，回答了同一个问题：
在商业世界里，到底该诚实，还是该有点心机？

第一个人：松下幸之助（诚实）
第二个人：菲尔·奈特（心机）

一边是亏到卖被子也要履约的松下。
一边是靠空壳公司一路骗出全球的奈特。
结局却殊途同归——两家企业都活过了百年。
```

#### 2. 时代对比

**写法**：
- 分析时代变化
- 解释为什么规则变了
- 用具体案例说明

**示例**：
```
如果把时钟拨回1962年，奈特的选择可能是对的。
那时候：
- 没有互联网，鬼冢无法在10分钟内查到"蓝带体育"是否存在
- 没有LinkedIn，日本高管无法验证奈特的履历

但今天，游戏规则彻底变了。
在互联网时代，撒谎就像在玻璃房子里偷东西。
```

#### 3. 结果对比

**写法**：
- 诚实 vs 谎言
- 长期 vs 短期
- 用数据对比

**示例**：
```
巴菲特：2008年说"我也不知道"，10年后300%回报
安然CEO：2001年说"商业模式完美"，6个月后破产
```

## 五、金句提炼法（增强记忆点）

### 核心原则

**每500-800字一个金句，让读者有记忆点**

### 金句类型

#### 1. 核心洞察金句

**特点**：揭示本质，让人"原来如此"

**示例**：
- "真诚不是道德光环，而是一种低成本、高复利、可持续的人生算法"
- "时间是最精确的称重机"
- "信任，是时间唯一无法伪造的货币"

#### 2. 比喻金句

**特点**：用生动的比喻，让抽象概念具体化

**示例**：
- "撒谎就像在玻璃房子里偷东西"
- "信用是利息最高的存款，只是兑现周期有点长"

#### 3. 行动指引金句

**特点**：直接告诉读者该怎么做

**示例**：
- "我想赢一次，还是赢一直？"
- "你是在和时间做朋友，还是做敌人？"

### 金句位置

- **开头**：1-2个，吸引注意力
- **中间**：每500-800字一个，保持节奏
- **结尾**：1-2个，强化记忆

## 六、可操作方法（提供实用价值）

### 核心原则

**不是空洞的说教，而是可操作的决策框架**

### 方法模板

#### 1. 问题清单法

**写法**：
- 用编号列出问题
- 每个问题有具体说明
- 提供判断标准

**示例**：
```
在做每个决定之前，问自己三个问题：

1. 这是一次性博弈，还是长期关系？
   - 如果是一次性（比如跟陌生人的商业谈判），适度的"策略性模糊"可以接受
   - 如果是长期关系（同事、伴侣、合伙人），任何谎言都会成为定时炸弹

2. 对方有多容易验证我说的话？
   - 如果对方10分钟就能查证（学历、婚史、公司业绩），撒谎等于自杀
   - 如果信息高度不对称，你有6个月的缓冲期

3. 我能承受最坏的后果吗？
   - 如果谎言被揭穿，代价是什么？
   - 如果最坏后果超过你的承受能力，就不要冒险。
```

#### 2. 决策框架法

**写法**：
- 提供判断标准
- 给出具体场景
- 说明适用条件

**示例**：
```
如果你想赢一次，心机可能是捷径。
但如果你想赢一直，真诚是唯一的路。
```

## 七、结尾写法（首尾呼应法）

### 核心原则

**回到开头故事，形成完整闭环**

### 写法模板

```
回到开头故事
  ↓
引用关键人物的话
  ↓
提炼核心观点
  ↓
用比喻强化记忆
  ↓
留下思考问题
```

### 示例分析

**原文结尾**：
```
2022年9月，82岁的乔伊纳德宣布把Patagonia全部股份捐给环保基金会。
记者问他："您不后悔吗？那可是30亿美元。"
老人笑了：
"如果我当年在黑色星期五打折促销，
也许能多卖几千件夹克，多赚几百万美元。
但我不会得到今天的Patagonia。
**因为真正的品牌，不是建立在交易上，而是建立在信任上。**
而信任，是时间唯一无法伪造的货币。"

所以，下次当你纠结该不该撒个"善意的谎言"时，
不妨想想：
**你是在和时间做朋友，还是做敌人？**
```

**结构拆解**：
1. **回到开头故事**：Patagonia、乔伊纳德
2. **引用关键人物的话**：老人的回答
3. **提炼核心观点**：信任是时间唯一无法伪造的货币
4. **留下思考问题**：你是在和时间做朋友，还是做敌人？

## 八、语言风格（生动不装腔作势）

### 核心原则

**用日常语言，不装腔作势，有温度**

### 语言技巧

#### 1. 口语化表达

**示例**：
- "董事会疯了"
- "手机炸了"
- "肾上腺素飙升"

#### 2. 具体数据

**示例**：
- 时间：2011年11月25日、1907年、1962年
- 金额：12万美元、30亿美元、1200亿美元
- 比例：30%增长、300%回报

#### 3. 细节描写

**示例**：
- "七岁时就能察言观色：奶奶喜欢学习好的，她就捧着课本"
- "当晚，他飞回俄勒冈，租两间空房，挂招牌、拍照、寄回日本"

#### 4. 对话引用

**示例**：
- "如果我们的生意建立在欺骗之上，那这门生意不做也罢。"
- "我也不知道。"

## 九、段落组织法（节奏控制）

### 核心原则

**长短结合，张弛有度，保持节奏**

### 段落类型

#### 1. 短段落（强调重点）

**示例**：
```
**销售额暴涨30%。**

五年后，Patagonia成为全球增长最快的户外品牌。
```

#### 2. 长段落（展开论述）

**示例**：
```
2011年11月25日。
黑色星期五。全美最疯狂的购物节。
```

## 十二、核心原则总结

### 1. 故事驱动

**用故事开头，用故事结尾，用故事贯穿全文**

### 2. 案例丰富

**建立多维度、多层次的案例体系**

### 3. 对比论证

**用对比让读者思考更深层的问题**

### 4. 金句频出

**每500-800字一个金句，让读者有记忆点**

### 5. 可操作性强

**不是空洞的说教，而是可操作的决策框架**

### 6. 首尾呼应

**回到开头故事，形成完整闭环**

### 7. 语言生动

**用日常语言，不装腔作势，有温度**

## 十三、应用优先级

### 高优先级（必须应用）

0. ✅ **标题命名**：反常识/冲突感 + 暗示核心观点 + 制造悬念
1. ✅ **震撼故事开头**：用反常识的故事开头
2. ✅ **首尾呼应**：回到开头故事
3. ✅ **金句提炼**：每500-800字一个金句
4. ✅ **可操作方法**：提供问题清单或决策框架

### 中优先级（尽量应用）

5. ✅ **历史对比案例**：建立深度
6. ✅ **个人化案例**：增加温度
7. ✅ **时代变化分析**：解释为什么

### 低优先级（可选应用）

8. ✅ **权威案例对比**：强化观点
9. ✅ **口语化表达**：增加生动性

## 十四、注意事项

### 1. 不要过度堆砌案例

**原则**：每个案例都要有明确的目的，不要为了案例而案例

### 2. 不要空洞的说教

**原则**：每个观点都要有案例支撑，每个方法都要可操作

### 3. 不要装腔作势

**原则**：用日常语言，有温度，不装逼

### 4. 不要忘记数据

**原则**：用具体数据支撑观点，让读者信服

### 5. 不要忘记首尾呼应

**原则**：开头和结尾要形成完整闭环
