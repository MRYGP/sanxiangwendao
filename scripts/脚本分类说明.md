# Scripts 目录脚本分类说明

## 📋 脚本分类

### ✅ 核心功能脚本（需要保留）

这些脚本是RAG系统和知识库维护的核心工具：

1. **build_index.py** - 构建向量索引
   - 功能：批量加载核心文档，进行分块、向量化，存储到向量数据库
   - 使用场景：首次构建索引或重置索引
   - 状态：✅ 保留

2. **test_query.py** - 测试查询
   - 功能：测试RAG系统的检索和问答功能
   - 使用场景：验证检索效果、调试查询
   - 状态：✅ 保留

3. **update_index.py** - 更新索引
   - 功能：增量更新单个或部分文档的索引
   - 使用场景：文档修改后更新索引
   - 状态：✅ 保留

4. **check_readme_links.py** - 检查README链接
   - 功能：检查README文件中的链接是否有效
   - 使用场景：维护文档链接完整性
   - 状态：✅ 保留

5. **reorganize_root_docs.py** - 重组文档
   - 功能：重组根目录文档
   - 使用场景：文档结构重组
   - 状态：✅ 保留

7. **batch_fix_commits.py** - 批量修复提交脚本
