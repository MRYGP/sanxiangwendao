# AI产品竞品分析系统

## 定位

系统化拆解 AI 产品与竞品，为“我们的产品迭代”提供可执行的洞察、建议与追踪机制。

## 产出物

- 竞品档案：按产品归档的 Step 0-10 分析文档
- 横向对比：赛道/功能维度的对比矩阵与结论
- 改进清单：可执行的产品优化建议（按优先级分层）
- 持续追踪：月度/季度/年度的竞品动态监控与预警

## 目录导航

- `00_系统指令/`：提示词与协作规则（基于 v2.0 拆分整理）
- `01_竞品库/`：按产品存放 Step 0-10 的拆解与报告
- `02_对比分析/`：按赛道/功能进行横向对比
- `03_追踪记录/`：月度追踪、季度报告
- `04_改进清单/`：从竞品分析中沉淀的行动建议
- `05_我们的产品/`：我们的产品基准资料（对比参照系）
- `06_分析工具箱/`：模板、清单、数据源

## 快速开始（建议顺序）

1. 在 `05_我们的产品/` 先写清楚：产品定位、核心功能、技术架构、用户画像
2. 在 `01_竞品库/` 复制 `产品模板/` 作为第一个竞品目录
3. 按 Step 0→10 执行分析；每步完成就落盘成文件并更新索引/追踪
