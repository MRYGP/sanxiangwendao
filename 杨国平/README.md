# README（杨国平思想体系文档集）

本目录包含 4 份关于“杨国平思想体系”的核心文档，用于**快速对齐认知**、**提取可复用框架**、以及**持续迭代更新**。

---

### 文档地图（4份文件各自负责什么）

- **`杨国平思想体系.md`**
  - **定位**：全景式“史官档案/总纲”
  - **你会得到**：起源动机 → 原创理论 → 产品架构 → 合作策略 → 演进轨迹 → 实现路径
  - **适合**：第一次完整了解体系、写对外叙事、做长期规划

- **`杨国平思想体系核心框架 v1.1 - 理论架构与应用索引.md`**
  - **定位**：高保真压缩的“宪法/索引”（用于 5-15 分钟对齐）
  - **你会得到**：哲学内核（内共生 vs 外骨骼、反脆弱性检验）+ 方法论桥梁（专家智慧AI化执行）+ 三层进化理论（诊断/个体/群体）+ 应用导航（对不同合作者怎么讲）
  - **适合**：对外沟通、招募协作、快速复盘方向

- **`杨国平思考更新.md`**
  - **定位**：体系的“教科书式展开 + 验证路线”
  - **你会得到**：三层金字塔结构、关键概念更工程化的表达、以及更明确的实验/指标意识
  - **适合**：把理论变成产品指标与验证计划、补齐论证链条

- **`杨国平认知画像.md`**
  - **定位**：人物“认知操作系统/认知基因组”画像
  - **你会得到**：权威筛选机制、学习方式（视频+AI）、人机分工、认知流水线（异常探测→极端推演→具象化→验证→内化）、优势与风险点
  - **适合**：理解“杨国平如何思考”、做合作/沟通/产品个性化设计

### 推荐阅读顺序（按目标选择）

- **只想 10 分钟理解核心**
  - 读：`杨国平思想体系核心框架 v1.1 - 理论架构与应用索引.md`

- **想理解“他是怎样的人/如何思考”**
  - 读：`杨国平认知画像.md` → `杨国平思想体系核心框架 v1.1 - 理论架构与应用索引.md`

- **想把思想体系讲给别人（合作/融资/对外表达）**
  - 读：`核心框架 v1.1` → `思想体系.md`（补叙事与案例）→ `思考更新.md`（补验证与指标）

- **想落地成产品与实验**
  - 读：`思考更新.md` → `核心框架 v1.1` → `思想体系.md`（找产品模块与案例）

### 关键概念最小索引（用来“抓住主线”）

- **根本选择**：认知内共生 vs 认知外骨骼  
- **检验法则**：拿掉AI之后，人是否更强（反脆弱性检验）  
- **核心桥梁**：专家智慧AI化执行（重建专家智慧涌现的“认知场域”，而非模仿输出）  
- **三层结构**：问题诊断（隐形代码/路径依赖）→ 个体递归（认知复利、史官系统）→ 群体飞轮（模因传染、网络相变）  
- **人机分工**：人负责异常探测/价值判断/整合；AI负责具体化/多角度/验证/长期记忆

---

### 如何持续更新（建议）

- **新增洞察**：优先写入 `杨国平思考更新.md`（作为“增量日志/验证路线”）
- **框架稳定后再压缩**：把新增内容提炼进 `核心框架 v1.1` 的结构（保持其“5分钟索引”属性）
- **人物画像变化**：如果认知习惯/偏好/风险点发生变化，再更新 `杨国平认知画像.md`
- **全景档案**：当新增内容足以改变叙事主线，再回填到 `杨国平思想体系.md`
