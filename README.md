    - 关键词：护城河、商业模式、AI创业、信任

**02-shu/innovation/破坏式创新.md**

    - 关键词：破坏式创新、克里斯坦森、边缘市场、够用就好、向上破坏、亚马逊、拼多多、蜜雪冰城

**02-shu/innovation/价值链创新.md**
    - 核心：创新不只发生在用户端，价值链有四个环节可以创新：用户端（谁能买）、分发端（谁能被看见）、供给端（谁能卖）、基础设施端（怎么交付）。改写规则比做得更好更重要
    - 关键词：价值链创新、改写规则、用户端、分发端、供给端、基础设施端、字节跳动、AWS、聚合者

**02-shu/innovation/供应链数字化.md**
    - 核心：从"大单慢反"到"小单快反"，数据驱动的生产决策。SHEIN如何用7天打败Zara的2周，通过数据前置、柔性供应链、算法驱动实现小单快反
    - 关键词：供应链数字化、小单快反、数据驱动、SHEIN、柔性供应链、先测后产

**02-shu/innovation/精益创业.md**
    - 核心：创业不是赌博，而是科学实验——用最小成本验证最大假设。Build-Measure-Learn循环，MVP（最小可行产品）验证，快速转型
    - 关键词：精益创业、MVP、Build-Measure-Learn、验证性学习、转型、Dropbox、Instagram、Zappos、小米、字节跳动、拼多多

**02-shu/innovation/从0到1.md**
    - 核心：真正的创新是从0到1，而不是从1到N；垄断比竞争更健康。垄断四特征（专有技术、网络效应、规模经济、品牌）、秘密理论、幂次法则
    - 关键词：从0到1、垄断、竞争、秘密理论、幂次法则、彼得·蒂尔、Google、特斯拉、微信、Airbnb、Uber

**02-shu/innovation/规则制定者.md**
    - 核心：从规则遵守者到规则制定者的三级进化路径。核心框架：找到无主之地+建立四大能力（货币/法律/身份/制裁）+形成飞轮。强调规则制定者不是在占领市场，而是在创造秩序
    - 关键词：规则制定者、规则改写者、生态主权、四大能力、跨时空系统、BNB、Starlink、宗教
