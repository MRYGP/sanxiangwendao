## 🔍 如何使用速查表

### Claude使用指南

1. **接到写作任务时**：
   - 先查看速查表，找到相关理论和案例
   - 使用`project_knowledge_search`精确搜索（提供文档ID或关键概念）
   - 避免全仓库盲目搜索

2. **引用理论时**：
   - 在"何时引用"列找到匹配场景
   - 引用时提供文档ID（如DOC-S001）
   - 说明理论如何验证或对比当前案例

3. **生成Cursor指令时**：
   - 在"引用文档"部分列出文档路径
   - 说明每个文档用于论证什么
   - 不要粘贴文档内容，只提供路径

### Cursor使用指南

1. **收到Claude指令时**：
   - 查看速查表，了解引用文档的核心理论
   - 从指定路径读取文档内容
   - 确保引用准确、论证充分

2. **写作时**：
   - 参考"关键概念"和"典型案例"
   - 确保引用的理论与案例匹配
   - 保持与仓库其他文档的一致性
