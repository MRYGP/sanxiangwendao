## 📖 详细内容说明

### 一、AI与认知理论类（8篇）

#### 1.1 核心理论框架

1. **认知内共生理论：AI时代人类能力进化的根本路径选择.md**
   - 核心：AI作为"认知教练"内化训练人类认知结构，而非外置工具
   - 关键词：认知内共生、认知外骨骼、正BUFF效应

2. **双螺旋进化模型：人机认知协同的进化动力学理论.md**
   - 核心：理性内化路径 + 创造性涌现路径的双螺旋进化
   - 关键词：理性内化、创造性涌现、AI催化剂

3. **递归式认知进化：情境触发事件驱动的人类智能跃迁理论.md**
   - 核心：情境触发认知事件(CTCE)作为人类智能跃迁的最小单元
   - 关键词：CTCE、认知递归增强、集体智慧飞轮

4. **集体认知涌现的多维相变模型：创新生态系统的动力学机制与工程实现.md**
   - 核心：七维临界点同时达到才能产生超线性创新的相变
   - 关键词：多维相变、认知多样性、涌现催化剂密度

#### 1.2 AI应用与实现

5. **AI史官集合论：三AI联合深度解析与完整实现方案.md**
   - 核心：AI作为"认知外挂"克服人类记忆、认知隧道、情绪偏见
   - 关键词：AI史官、三层技术架构、时间护城河

6. **专家智慧AI化执行理论：从认知场域到智慧共振的哲学与科学框架.md**
   - 核心：将人类"认知场域"转化为AI可执行的"智慧共振系统"
   - 关键词：认知场域、智慧共振、认知疫苗、认知冲突时刻

7. **AI驱动的创新工程系统.md**
   - 核心：InnoSynapse框架，将不确定的创新过程系统化、工程化
   - 关键词：Foundation、Exploration、Simulation、Validation、Collaboration

8. **AI 在金融分析中的核心价值——发现人脑无法触及的"暗规律"非线性，非逻辑，非理性（人脑偶尔却能超越AI的发现).md**
   - 核心：AI发现非线性、非逻辑、非理性的"暗规律"
   - 关键词：暗规律、探索工具、效率工具、人机协作
