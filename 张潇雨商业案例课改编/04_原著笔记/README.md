# 04_原著笔记

本目录用于存放课程/原材料的**忠实提炼**（便于回溯证据链）。  
定位：它不是“分析输出”，而是“可追溯的来源笔记”。

---

## 你应该把什么放在这里

- 课程要点的结构化摘录（最好带时间戳）
- 关键数据/引用的出处与上下文
- 术语解释（以课程口径为准）
- 你为“待验证点”补的公开资料摘要（注明来源）

---

## 组织规则

- 每个章节/期数一个文件，文件名建议：
  - `第X期-主题.md` 或 `章节X-主题.md`
- 每条摘录尽量包含：
  - 原话/要点
  - 时间戳或段落位置
  - 你认为它支持的“结论/模型/洞察”（可选）

---

## 与案例/模型/洞察的关系

- 案例文档里的关键事实，尽量能回指到本目录某一条摘录
- 模型与洞察的“证据链”优先引用本目录的出处
