# 01_案例库

本目录存放**单个案例**的标准化拆解文档。  
原则：**同一份案例只保存一处**（建议放在“按行业分类”），其余分类通过“标签 + 索引”实现快速检索，避免多处复制导致版本漂移。

---

## 目录结构说明

- `按行业分类/`：推荐的主落盘位置（最直观）
- `按商业模型分类/`：用于按模型视角组织（目录可为空，靠索引与链接使用）
- `按企业生命周期分类/`：用于按阶段检索（目录可为空，靠索引与链接使用）
- `案例总索引.md`：全局入口（必维护）

---

## 新增一个案例的标准流程

1. 先用 `00_系统指令/Claude-案例拆解提示词.md` 输出结构化要点
2. 选择一个落盘路径（建议：`按行业分类/<行业>/`）
3. 创建案例文件并补齐元信息（来源、标签、日期）
4. 更新 `案例总索引.md`
5. 同步更新 `05_学习记录/学习进度追踪器.md`

---

## 命名规范

- `<企业名>-<案例关键词>.md`
- 关键词建议：转型/冷启动/会员制/定价/渠道/组织/规模化等
